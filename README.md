# Projects in Mathematical and Computational Methods in Chemical Engineering (IQ260-Unicamp)

Repository link: https://github.com/LeonardoUnicamp/Projects_in_Mathematical_and_Computational_Methods_in_Chemical_Engineering
Projects executed in the GNU Octave environment (version 10.2.0)

## First Project
Project focused on solving a problem in the theme of “Numerical Differentiation and Integration,” with the development of a 4th-degree polynomial for numerical resolution in comparison with the Trapezoidal Rule, Simpson’s 1/3 and 3/8 methods, and their multiple applications.

## Second Project
Project focused on solving a problem in the theme of “Systems of Linear Equations,” with the application of Cramer’s method, Gaussian Elimination, and the Inverse Matrix.

## Third Project
Project focused on solving a problem in the theme of “Nonlinear Systems of Equations,” with the application of the Newton–Raphson method.

## Fourth Project
Project focused on solving a problem in the theme of “Ordinary Differential Equations (ODEs),” with the application of the Euler, Predictor–Corrector, and 4th-order Runge–Kutta (RK4) methods, using data obtained from the article “Mathematical model of growth and polyhydroxybutyrate production using microbial fermentation of Bacillus subtilis,” by Ipsita Panda, Soumyajit Balabantaray, Subhendu Kumar Sahoo, and Nivedita Patra (DOI: 10.1080/00986445.2017.1384923).

## Fifth Project
Project focused on solving a problem in the theme of “Partial Differential Equations (PDEs),” with the application of the Gauss–Seidel (or Liebmann) method for solving the Laplace Equation and Fourier’s Heat Equation for a metallic plate.

## Sixth Project
Project for a seminar presentation, with the resolution of Example 8.7, pages 293–294, from the book “Elements of Chemical Reaction Engineering” by H. Scott Fogler (6th edition), applying the 4th-order Runge–Kutta (RK4) method.

# Projetos de Métodos Matemáticos e Computacionais para Engenharia Química (IQ260-Unicamp)

Link do repositório: https://github.com/LeonardoUnicamp/Projects_in_Mathematical_and_Computational_Methods_in_Chemical_Engineering
Projetos executados em ambiente de GNU Octave (versão 10.2.0)

## Primeiro Projeto
Projeto focado em resolver uma problemática no tema de "Derivação e Integração Numérica", com o desenvolvimento de um polinômio de grau 4 para resolução numérica em comparação com os métodos de Regra do Trapézio, Simpson 1/3 e 3/8 e suas aplicações múltiplas.

## Segundo Projeto
Projeto focado em resolver uma problemática no tema de "Sistemas de Equações Lineares", com aplicação dos métodos de Cramer, Eliminação Gaussiana e Matriz Inversa.

## Terceiro Projeto
Projeto focado em resolver uma problemática no tema de "Sistemas de Equações Não Lineares", com aplicação do método de Newton-Raphson.

## Quarto Projeto
Projeto focado em resolver uma problemática no tema de "Equações Diferenciais Ordinárias (EDOs)", com aplicação dos métodos de Euler, Preditor-Corretor e Runge-Kutta de ordem 4 (RK4), a partir dos dados obtidos no artigo "Mathematical model of growth and polyhydroxybutyrate production using microbial fermentation of Bacillus subtilis", de Ipsita Panda, Soumyajit Balabantaray, Subhendu Kumar Sahoo e Nivedita Patra (DOI: 10.1080/00986445.2017.1384923).

## Quinto Projeto
Projeto focado em resolver uma problemática no tema de "Equações Diferenciais Parciais (EDPs)", com aplicação do método de Gauss-Seidel (ou Liebmann) para resolução da Equação de Laplace e da Equação de Calor de Fourier para uma placa metálica.

## Sexto Projeto
Projeto para apresentação de seminário, com resolução do Exemplo 8.7, da página 293-294, do livro "Elemento de Engenharia das Reações Químicas" de H. Scott Fogler (6th edição), aplicando o método de Runge-Kutta de ordem 4 (RK4).
